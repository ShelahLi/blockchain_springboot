# block
区块链电子证据保全项目
参考网址： ybear-web.com

开发环境：
1. JDK1.8
2. Maven
3. Spring Boot + SpringMVC + Mybatis
4. Spring Security + Spring Social + Spring OAuth
5. Bubi SDK
6. web3j
7. EVM

---

目标： 
1. 提供基于Spring Security 的安全网站
2. 提供常规登录模式以及第三方登录
3. 基于布比区块链完成数据模块的搭建，保证证据信息上链
4. 基于web3j实现对以太坊的链上代码调用
5. 基于配置选择布比或者以太坊进行数据上链服务
5. 对于热点数据采用Redis进行保存
6. 尽力完成基于证据链的系统开发
